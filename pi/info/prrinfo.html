Priority Round-robin scheduling

Priority Round-robin is an algorithm in which the operating system uses a timer, called quantum along with the priority selection. The priorities may be assigned based upon deadline,burst time or other criteria.

  • Advantages
      • Simplicity;
      Priority based round robin has less number of context switches in comparison to simple round robin for same value of time quantum.
      Generates time slot for each process,making the process queue does not stand still in a single process.
      Tasks may be prioritized.
  • Disadvantages
      • Problem of Starvation may occure.
      Not a fair Scheduling.
