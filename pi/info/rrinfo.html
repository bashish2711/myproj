Round-robin scheduling

Round-robin is an algorithm in which the operating system uses a timer, called quantum, where all processes gain the same amount of quantum to rotate in the CPU.

  • Advantages
      • Simplicity;
      Easy to implement;
      Generates time slot for each process,making the process queue does not stand still in a single process.
      Each process gets equal CPU time so no process is served with priority, Hence fair scheduling.
      No problem of starvation.
  • Disadvantages
      • Diffcult to select the time quantum.
      If time quantum is large this becomes same as First Come First Serve Scheduling.
      If time quantum is small the context switching and hence the processor overhead is large.
